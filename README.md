# 🤖 Telegram Bot для обучения основам JavaScript

Телеграм-бот для интерактивного изучения JavaScript с нуля. Бот предоставляет уроки, практические упражнения, отслеживание прогресса и персонализированные рекомендации.

## ✨ Возможности

- 📚 **Уроки по JavaScript** — структурированные уроки от основ до продвинутых тем
- ✅ **Практические упражнения** — интерактивные задания с проверкой кода
- 📊 **Отслеживание прогресса** — статистика по завершенным урокам и упражнениям
- 🎯 **Персонализация** — адаптация под уровень пользователя
- 🔄 **Интерактивное обучение** — пошаговое прохождение уроков с навигацией
- 🏆 **Система достижений** — очки и награды за успехи

## 🏗️ Архитектура

Проект построен на современном стеке технологий:

- **Python 3.13+** — основной язык разработки
- **Aiogram 3.3** — асинхронный фреймворк для Telegram Bot API
- **SQLAlchemy 2.0** — ORM для работы с базой данных
- **SQLite** — база данных (можно заменить на PostgreSQL)
- **Redis** — кэширование и управление состояниями (опционально)

## 🚀 Быстрый старт

### 1. Клонирование и настройка

```bash
git clone <репозиторий>
cd js-learning-bot
```

### 2. Создание виртуального окружения

```bash
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

Отредактируйте `.env`:

```env
BOT_TOKEN=your_telegram_bot_token_here
DATABASE_URL=sqlite+aiosqlite:///db.sqlite3
REDIS_URL=redis://localhost:6379/0  # опционально
```

**Как получить BOT_TOKEN:**
1. Откройте Telegram и найдите @BotFather
2. Создайте нового бота: `/newbot`
3. Скопируйте выданный токен

### 5. Инициализация базы данных

```bash
python init_db.py
```

Скрипт создаст таблицы и заполнит их демо-данными:
- 3 урока по JavaScript (начальный уровень)
- 7 практических упражнений

### 6. Запуск бота

```bash
python main.py
```

Бот начнет работать и будет отвечать на сообщения в Telegram.

## 📖 Использование бота

### Основные команды

- `/start` — начать работу с ботом, регистрация
- **📚 Уроки** — просмотр доступных уроков
- **✅ Практика** — выполнение упражнений
- **📊 Прогресс** — статистика обучения
- **⚙️ Настройки** — настройки профиля (в разработке)

### Процесс обучения

1. **Регистрация** — при первом запуске бот запросит имя, уровень подготовки и цель обучения
2. **Выбор урока** — пользователь выбирает урок из меню
3. **Изучение** — постраничное изучение материала
4. **Практика** — выполнение упражнений после урока
5. **Прогресс** — отслеживание завершенных уроков и статистики

### Типы упражнений

- **Кодовые задания** — написание JavaScript кода
- **Множественный выбор** — выбор правильного ответа из вариантов
- **Текстовые ответы** — ввод текстового ответа

## 🗄️ Модели данных

### User (Пользователь)
- `telegram_id` — уникальный ID пользователя в Telegram
- `name` — имя пользователя
- `level` — уровень подготовки (beginner, intermediate, advanced)
- `goal` — цель обучения
- `is_active` — активен ли пользователь

### Lesson (Урок)
- `title` — название урока
- `description` — краткое описание
- `content` — содержимое урока (Markdown)
- `order` — порядковый номер
- `difficulty` — сложность (beginner, intermediate, advanced)
- `estimated_time` — предполагаемое время прохождения

### Exercise (Упражнение)
- `lesson_id` — ссылка на урок
- `question` — вопрос/задание
- `code_template` — шаблон кода (если требуется)
- `correct_answer` — правильный ответ
- `answer_type` — тип ответа (code, multiple_choice, text)
- `options` — варианты ответов для multiple_choice (JSON)
- `explanation` — объяснение правильного ответа
- `points` — очки за выполнение

### CompletedLesson / CompletedExercise
- Отслеживание прогресса пользователя
- Время завершения, количество попыток, результат

### UserProgress
- Общая статистика пользователя
- Количество завершенных уроков/упражнений
- Общее количество очков
- Дней подряд (streak)

## 🔧 Расширение функциональности

### Добавление нового урока

1. Отредактируйте `init_db.py`:
   - Добавьте новый словарь в список `lessons_data`
   - Укажите содержание урока в поле `content` (поддерживает Markdown)
   - Задайте порядок (`order`) и сложность (`difficulty`)

2. Добавьте упражнения для урока:
   - Добавьте словари в список `exercises_data`
   - Укажите `lesson_id` соответствующего урока
   - Определите тип упражнения (`answer_type`)

3. Переинициализируйте базу данных:
   ```bash
   python init_db.py
   ```

### Добавление нового типа упражнений

1. Расширьте модель `Exercise` в `models.py`:
   - Добавьте новый `answer_type`
   - При необходимости добавьте поля для хранения данных

2. Реализуйте обработку в `handlers/practice.py`:
   - Добавьте логику проверки нового типа ответа
   - Создайте соответствующую клавиатуру

3. Обновите `init_db.py` для создания упражнений нового типа
